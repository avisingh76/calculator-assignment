# Calculator & Grade Program

This project contains three Python programs.

## Stage 1: Basic Calculator
- Takes two numbers and an operator (+, -, *, /)
- Displays result
- Handles division by zero

## Stage 2: Calculator with Result Check
- Same as Stage 1
- Also checks if result is Positive, Negative, or Zero

## Stage 3: Student Grade Calculator
- Input: Student name and 3 subject marks (0–100)
- Calculates Total (out of 300) and Percentage
- Assigns Grade:
  - A: ≥ 75%
  - B: ≥ 60%
  - C: ≥ 40%
  - F: < 40%
